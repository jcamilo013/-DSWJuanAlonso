• OBJETIVO GENERAL DEL SITIO

    • Por medio de este sitio, buscamos que se el estudiante adquiera los conocimientes de CSS3: Estilos internos y Externos,Selectores CSS,
    permitiendole aplicarlos a los sitios web diseñados con el fin de mejorar su presentacion y entendimiento del sitio

    • Objetivos Especificos

      • Conocer los conceptos y etiquetado de CCS3, que es la última evolución del lenguaje de las Hojas de Estilo en Cascada,
        que nos sirve para definir el aspecto de las páginas web de una manera más avanzada.

      • Brindar la capacidad de aplicar estos estilos a los documentos HTML, ahorrarnos tiempo y trabajo al poder seguir varias técnicas
        sin necesidad de usar un editor gráfico.

      • Permitir un mayor control de la presentación de los sitios diseñados, al poder tener todo el código CSS reunido en uno, lo que facilita su modificación.

  • PRESENTACION DE LA SECCION

Hemos visto que hay un estilo predeterminado, cuando no se especifica ningún estilo, el navegador web aplicará automáticamente el estilo predeterminado
y podemos incrustar el código CSS en la propia marca HTML "en línea". Ahora, veremos otras dos formas de aplicar estilos CSS: en la parte inicial del
documento HTML (estilo interno o CSS interno) o en un archivo .css separado (estilo externo o CSS externo) del archivo HTML.
